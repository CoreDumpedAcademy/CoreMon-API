# CoreMon-API
Api for coremon

This API will be used to access to the game's database which will store:

* Save states linked to an account
  * User info
  * Bag items
  * Money
  * Coremon Team
  * Coredex
* Coremon's base stats and attacks
* Moves' data

Information will be retreived and stored as it becomes necessary.
* Store:
  * Save game
* Retrive:
  * Load game
  * Load wild coremon
